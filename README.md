A fun and interactive movie recommendation app built using **MERN** and **OpenAI API**.

Make sure you have installed:

1. Node.js (v18 or higher)

2. git clone https://github.com/your-username/your-repo-name.git

3. cd Frontend
4. npm install

5. create .env
6. paste this in the .env
     VITE_BASE_URL=http://localhost:5000/api/
7. npm run dev
